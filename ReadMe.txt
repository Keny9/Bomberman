
*******************BOMBERMAN*******************

Pour se déplacer, utiliser les touches W A S D
Pour poser une bombe, appuyer sur la barre espace
Lorsque la deuxième bombe est disponible, utiliser la touche E pour la poser
Pour quitter le jeu à n'importe quel moment, appuyer sur la touche ESC

Tuer un ennemi : 100 pts
Détruire un mur : 50 pts

Bomberman a 5 vies, représentées par les coeurs.
Lorsqu'il vient de subir du dommage, il devient invincible pendant 5 secondes, représenté par l'étoile.
